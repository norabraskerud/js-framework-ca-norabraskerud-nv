### Installation

1. Clone the repo:

```bash
git clone git@github.com:norabraskerud/js-framework-ca-norabraskerud-nv.git
```

2. Install the dependencies:

```
npm install
```

### Running

Run development server

```bash
npm run start
```

Build

```bash
npm run Build
```
